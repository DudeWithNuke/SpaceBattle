# Grid

## Definition
Grid — трёхмерное игровое пространство, состоящее из набора Cell, в котором размещаются корабли игрока.

Каждый игрок имеет **собственный Grid**.  
Игровые поля игроков логически изолированы и скрыты туманом войны.

---

## Structure
Grid представляет собой:
- трёхмерную сетку `Cell[x, y, z]`
- фиксированный размер по осям `X`, `Y`, `Z`
- независимую систему координат для каждого игрока

Grid логически разделён на слои по оси `Y`.

Навигация по слоям осуществляется с помощью Cursor Plane.

---

## Core Properties
- `width`, `height`, `depth`
- список всех Cell
- список размещённых кораблей

---

## Rules
- В одной Cell может находиться только одна часть корабля.
- Корабли занимают набор **соседних** Cell согласно форм-фактору корпуса.
- После начала боя корабли не перемещаются.

---

## Gameplay Role
Grid является:
- контейнером игровых объектов
- контекстом для всех боевых расчётов
- источником данных для fog of war

---

## Technical Notes
Grid не является отдельной физической сценой Unity.  
Оба игровых поля существуют в одной сцене и различаются логически.
