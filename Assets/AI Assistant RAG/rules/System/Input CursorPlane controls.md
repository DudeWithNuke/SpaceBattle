# Cursor Plane Controls

## Перемещение между слоями

**СКМ (прокрутка колеса мыши)**
- Перемещает Cursor Plane вверх или вниз по оси Y
- Камера перемещается вместе с Cursor Plane, сохраняя относительную дистанцию и угол обзора

Таким образом игрок последовательно просматривает разные горизонтальные срезы 3D Grid.
