# Placeable Object — Управление

## ЛКМ (Левая кнопка мыши)
Используется для взаимодействия с Placeable Object:

### Подбор объекта
- Игрок может держать **только один** Placeable Object одновременно
- Подбор возможен **с любого слоя Grid**, независимо от положения Cursor Plane
- Объекты можно подбирать даже с ячеек в состоянии `DisabledLayer`

После подбора:
- Объект переходит в состояние **Picked**
- Объект автоматически выравнивается по уровню текущего Cursor Plane

### Размещение объекта
- Размещение возможно **только на ячейках ActiveLayer**
- После размещения объект переходит в состояние **Placed**

---

## Центр захвата объекта
При подборе объект привязывается к курсору **по своему геометрическому центру**, а не по точке клика.
Пример: если размер объекта `3x2x1`, то точка привязки будет находиться в центре его объёма.
