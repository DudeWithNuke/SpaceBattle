# Player Input System — Overview
Данный документ описывает базовое управление игрока в боевом режиме без учета UI.

Система ввода управляет:
- размещением объектов (Placeable Object)
- камерой
- навигацией по слоям Grid через Cursor Plane
- переключением между игровыми полями игроков
