# Reflex — Installation

## Unity Package Manager
1. Открой Window → Package Manager  
2. Нажми + → Add package from git URL...  
3. Вставь: https://github.com/gustavopsantos/reflex.git?path=/Assets/Reflex/#13.0.3
или в качестве OpenUPM:

## UnityPackage
1. Скачать `.unitypackage` из релизов  
2. Импортировать в проект

После установки библиотека доступна в коде через namespace `Reflex.Core`.
